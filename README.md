## Start Here

This is a template Validated Pattern for installing and using Ansible Automation Platform on OpenShift. It uses
a Config as Code (CaC)/GitOps approach to loading the AAP configuration.

The concepts in this pattern were developed for and derived from the Ansible Edge GitOps Validated Pattern.
If you've followed a link to this repository, but are not really sure what it contains
or how to use it, head over to [Ansible Edge GitOps](https://validatedpatterns.io/patterns/ansible-edge-gitops/)
for additional context and installation instructions.
